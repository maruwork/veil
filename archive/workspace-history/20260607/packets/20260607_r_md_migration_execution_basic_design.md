# r.md Migration Execution Basic Design

## 1. Reclassification Rationale

### 必須

- `readback`
  - verification / report close の中心語

### 推奨

- `root entry`
  - governance entry の統一価値がある
- `runtime copy`
  - runtime 文脈での統一価値がある

### 観察

- `race condition`
  - 一般技術語で広い
- `redundant`
  - 一般形容語で広い

## 2. Write Shape

```md
# r

## 必須

- readback → 読み戻し確認

## 推奨

- root entry → ルートエントリー
- runtime copy → ランタイムコピー

## 観察

- race condition → 競合状態
- redundant → 冗長な
```
