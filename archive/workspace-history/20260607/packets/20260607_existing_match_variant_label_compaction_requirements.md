# Requirements

## Goal

`existing-match` の variant 表示ラベルを短くし、`README.md` に残る旧 detail label 文言も current 契約へ戻す。

## Scope

- `veil-normalize.py` の `existing-match` text 出力
- `README.md`
- `docs/veil-design.md`
- `skills/codex/veil-capture/SKILL.md`
- `skills/claude-code/veil-capture.md`
- `index/project-current-work.md`

## Acceptance

- `existing-match` の variant 表示が `| variants: ...` に短縮される
- `README.md` の旧 `選別/保留/判別` / `選別/判別` 表記が消える
- code / README / design / skills / current companion が同じ契約で一致する
