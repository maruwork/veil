# p.md Migration Execution Basic Design

## 1. Reclassification Rationale

### 必須

- `pending`
  - 保留運用の中核語

### 推奨

- `people first`
  - 統一価値はあるが hard gate 中心語ではない

### 観察

- `path`
  - 機械処理語と概念語が衝突しやすい
- `pipeline`
  - 一般技術語で文脈依存が強い
- `portability split`
  - 低頻度かつ project 文脈依存

## 2. Write Shape

```md
# p

## 必須

- pending → 保留

## 推奨

- people first → ピープルファースト

## 観察

- path → 道のり
- pipeline → パイプライン
- portability split → 持ち出し分離
```
