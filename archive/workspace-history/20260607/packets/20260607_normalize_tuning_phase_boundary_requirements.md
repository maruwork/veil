# Requirements

## Goal

`normalize` tuning を無限継続させないため、現在の停止線と次 phase の handoff を current surface に固定する。

## Scope

- `index/project-current-work.md`
- `README.md`
- `docs/veil-design.md`

## Acceptance

- `normalize` tuning の到達点と停止線が current surface で読める
- 次は `capture` 候補抽出 tightening へ進むと明記される
- 同種の micro-tuning は「具体的な誤判定が出た時だけ再開」と読める
