概要

VEIL は AI が生成した英語テキストを、あなたの語彙ルールに沿った日本語に変換するローカルツールです。 語彙ルールの正本は ~/.veil/rules/ に保存されます。

起動python app.py → http://<ホスト>:8080
依存Python 3.8 以上、標準ライブラリのみ
正本データ~/.veil/rules/（語彙ルールの正本）
補助データ{repo}/vocab.db（SQLite、UI 管理用）
新規 vocab.db には、少数の高需要・高確度語だけが helper seed として入ります。完全な辞書ではなく、既存 vocab.db を自動で掃除するものでもありません。

既存 vocab.db の見直し材料が必要な時は、非破壊の監査補助を使います。

python veil-audit-db.py
python veil-audit-db.py --status drop-candidate
python veil-audit-db.py --status review
python veil-audit-db.py --json

棚卸しでは、まず drop-candidate を見て、次に review を確認すると扱いやすいです。

  • drop-candidate: 不要なら語彙一覧の × ボタン で削除を検討する
  • review: 候補1やカテゴリを見直し、必要なら編集フォームやポップアップ選択で上書き更新する
  • keep: 通常はそのまま維持する

語彙一覧には、各行の監査バッジ、件数サマリ、status 絞り込み、棚卸し優先 の並び替えが表示されます。review と drop-candidate の行には、見直し焦点も薄く表示されます。削除候補 は確認つきで一括削除でき、confirm に対象語 preview も出ます。要見直し ボタンでは、先頭候補を編集フォームへ送れ、現在編集中の review 候補も分かります。保存や削除でその語が review ではなくなった場合だけでなく、別の語の編集へ移った場合も、この表示は自動で外れます。

基本の使い方

テキスト入力

画面上部のテキストエリアに AI が出力したテキストを貼り付けます。

変換実行

  1. テキストを貼り付ける
  2. 変換する ボタンをクリック（または Ctrl+Enter）
  3. 左列に元テキスト、右列に変換後テキストが表示される
  4. 変換された語は 緑色でハイライト 表示される

変換候補の変更

ハイライト語をクリックするとポップアップが開きます。

操作説明
候補を選択その変換語を採用。選択した語が次回から候補1になる
自分で設定入力欄に直接入力して確定
元に戻す変換を取り消して元の英語に戻す（DB は変更しない）
変換しない（対象外へ）この語を変換対象から除外。以後スキップ

語彙の登録

フォームから登録

サイドバー上部の「語彙を追加」フォームを使います。

フィールド説明
元の語（英語）変換対象の英単語・フレーズ。必須
候補1採用語。実際の変換に使われる語
候補2（任意）代替候補。候補として保持されるが同期はされない
候補3（任意）予備候補。候補として保持されるが同期はされない
カテゴリ下の「カテゴリ」を参照

元の語を入力すると、カテゴリが自動推定されます。翻訳候補は手動で入力してください。

追加 / 更新 を押すとボタンが一時的に緑に変わり、登録完了を通知します。 同じ元語を再登録すると上書き更新（upsert）されます。

クイック登録（テキスト選択）

  1. 変換後テキストをマウスでドラッグ選択
  2. 選択語の近くに + 登録 ボタンが浮き上がる
  3. クリックすると登録フォームに自動入力

未変換語リストから登録

変換実行後、サイドバーの「未変換語」セクションに変換されなかった英単語がチップで表示されます（最大10件、出現回数順）。 チップをクリックすると登録フォームに自動入力されます。

ストップワード（a, the, is など一般的な英単語）と登録済みの語は除外されます。

登録済み語彙

サイドバー下部の「登録済み」リストで登録語を確認・管理できます。

操作説明
検索ボックス元語または候補1で絞り込み。最大20件表示
カテゴリフィルタ特定カテゴリのみ表示
頻度 / 登録順 ボタンソート順を切り替え
× ボタン語彙を削除

表示の見方

表示状態意味
通常（不透明）変換対象の有効エントリ
薄い（半透明）cat=2（対象外）または翻訳なし。変換に使われない
(翻訳なし)変換対象カテゴリだが翻訳が未入力。複合フレーズの保護用途や入力待ち

Export / Import

フッター右側のボタンから語彙データをバックアップ・移行できます。

Export

現在の語彙DBに登録されている語彙をすべて JSON ファイルとしてダウンロードします。

ファイル名: veil-vocab-YYYYMMDD.json

Import

JSON ファイルを選択して読み込みます。既存の語彙は upsert（上書き更新）されます。 削除はされません。読み込んだ語彙は現在の語彙DBに登録されます。

Import ファイルの必須フィールドは o（元語）のみです。候補1がない語は翻訳なしで登録されます。

veil-sync — 他の AI ツールへの語彙同期

~/.veil/rules/ にある語彙ルールを CLAUDE.md, AGENTS.md, .cursorrules などの設定ファイルに挿入します。 自動同期は行わず、必要なタイミングで veil-sync.py を実行します。

初期設定

python veil-sync.py --add /path/to/CLAUDE.md

登録と同時に即時同期されます。複数ファイルを登録する場合は繰り返し実行します。

コマンド一覧

コマンド説明
python veil-sync.py --add <path>同期先を登録
python veil-sync.py --list登録済み一覧を表示
python veil-sync.py --remove <path>登録を解除
python veil-sync.py手動で全ファイルを同期

対応ファイル

ツールファイル
Claude CodeCLAUDE.md
CodexAGENTS.md
Cursor.cursorrules
GitHub Copilot.github/copilot-instructions.md
Gemini CLIGEMINI.md
Aider.aider.conf.yml

設定ファイルに以下のブロックが挿入されます（.yml / .yaml は # 形式）。同期されるのは候補1だけです。候補2 / 候補3 は同期されません。

<!-- VEIL_START -->
以下の語彙ルールに従って出力してください：
- common asset → 共通資産
- current state → 今の状態
- validator → バリデータ
<!-- VEIL_END -->

変換されないケース

以下のパターンは意図的に変換をスキップします。

パターン例
ファイル名 / パスの一部（. 隣接）workflow_close.py, path/to/close.md
ハイフン区切り識別子（- 隣接）task-close-workflow
バッククォート内`close()`
ダブルクォート内"close"
key=value 形式status=close, type=branch
チケット ID（LETTERS-000）BRANCH-046, CLOSE-012
意図しないスキップが起きた場合は、「自分で設定」ポップアップで手動変換できます。

カテゴリ

cat名称変換対象用途
1説明語✓一般的な技術英語（close→完了 等）
2固定値・対象外—ALL_CAPS 定数、明示的に変換しない語
5固有名詞・製品名✓ツール名・ライブラリ名
6プロジェクト固有✓プロジェクト独自の用語
7境界が曖昧✓文脈によって変換が必要な語

元の語を入力すると、以下のルールでカテゴリが自動推定されます。

パターン推定カテゴリ
ALL_CAPS、UPPER_CASE、LETTERS-000、key=value、パス、拡張子付き名2（固定値・対象外）
大文字小文字が混在する語（camelCase、SQLite 等）5（固有名詞）
小文字中心の空白区切りフレーズ（current state 等）1（説明語）
小文字中心だがハイフン/アンダースコアを含む語、単語だけでは用途が定まりにくい語7（境界が曖昧）
この自動推定は最初の入口です。推定結果をそのまま一気に全部採用せず、識別子か、固有名か、説明語か、プロジェクト固有語か、境界が曖昧な語かを見直してから登録してください。特に project 固有 は自動では付けません。

採用は高需要の語から段階的に進めます。

優先順先に採用する語
1頻出で困り度が高い語
2VEIL 運用そのものに効く基幹語
3プロジェクト固有語
4低頻度語と境界が曖昧な語
4 は急いで採用せず、保留として次回以降に回してかまいません。固有名か一般語か迷う場合、既存ルールと別訳が衝突する場合、まだ困り度が低い場合も保留に回します。

Windows 自動起動

Windows ログイン時に VEIL サーバーを自動起動するタスクを登録できます。

python install-startup.py           # 登録
python install-startup.py --remove  # 解除
python install-startup.py --status  # 確認

登録後、次回 Windows ログイン時から自動で起動します。コンソールウィンドウは表示されません。
