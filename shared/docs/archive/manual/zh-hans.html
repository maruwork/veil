概述

VEIL 是一个本地工具，可将 AI 生成的英文文本转换为您偏好的词汇。转换规则存储在 SQLite 中。自动生成翻译候选词使用 DeepL 免费 API（API 密钥为可选项）。

启动python app.py → http://<主机>:8080
要求Python 3.8 以上，仅使用标准库
数据vocab.db（SQLite，首次运行时自动创建）

基本用法

文本输入

将 AI 生成的文本粘贴到屏幕顶部的文本框中。

执行转换

  1. 将文本粘贴到文本框中
  2. 点击转换（或按 Ctrl+Enter）
  3. 左栏显示原文，右栏显示转换后的文本
  4. 已转换的词语以绿色高亮显示

更改候选词

点击高亮显示的词语即可打开弹出窗口。

操作说明
选择候选词使用该翻译。下次所选词语将成为 p1（首选候选词）
直接输入直接输入并确认
还原撤销转换并恢复原始英文（不更改数据库）
排除将该词从转换中排除，此后将跳过

词汇注册

表单注册

使用侧边栏顶部的"添加词汇"表单。

字段说明
原词（英文）要转换的英文单词或短语。必填
译文 1主要翻译
读音（可选）发音或备用读法
译文 2（可选）第二候选词
类别请参阅类别部分

输入原词时，类别会自动检测。如果设置了 DeepL API 密钥，从弹出窗口或未转换词芯片打开表单时，翻译候选词将自动填入。

添加 / 更新按钮会短暂变为绿色以确认注册。重新注册相同单词将执行 upsert（覆盖更新）。

快速注册（文本选择）

  1. 在转换后的输出中拖动选择文本
  2. 选区附近会出现+ 注册按钮
  3. 点击即可自动填写表单并获取 DeepL 候选词

从未转换词注册

转换后，未被转换的英文词语会以芯片形式显示在侧边栏的"未转换词"部分（最多 10 个，按频率排序）。点击芯片即可自动填写注册表单。

停用词（a、the、is 等）和已注册词语将被排除。

已注册词汇

侧边栏底部的"已注册"列表可查看和管理词汇。

操作说明
搜索框按原词或译文 1 筛选。最多显示 20 条
类别筛选仅显示特定类别
频率 / 顺序按钮切换排序方式
× 按钮删除词汇条目

列表说明

状态含义
正常（不透明）转换中使用的活动条目
暗淡（半透明）cat=2（已排除）或无翻译。不用于转换
（无翻译）属于目标类别但未输入翻译。用作占位符或等待输入

导出 / 导入

使用页脚右下角的按钮备份和迁移词汇数据。

语言对与数据隔离

词汇数据按语言对独立管理。 EN→JA 和 EN→KO 的数据存储在同一个 vocab.db 中，但互不干扰。 导出和导入均仅针对当前选择的语言对。

导出

将当前语言对中注册的所有词汇下载为 JSON 文件。

文件名: veil-vocab-YYYYMMDD.json

导入

选择 JSON 文件加载。现有词汇将执行 upsert（覆盖更新）。不会删除任何内容。导入的词汇将注册到当前语言对中。

若要导入其他语言的数据，请先切换标题栏的语言选择器，再执行导入。
导入文件中唯一必填字段是 o（原词）。没有 p1 的词语将以无翻译方式注册。

veil-sync — 与其他 AI 工具同步

自动将已注册词汇插入 CLAUDE.md、AGENTS.md、.cursorrules 等配置文件。每次添加或删除词汇时都会在后台同步。

初始设置

python veil-sync.py --add /path/to/CLAUDE.md

注册后立即同步。对多个文件重复操作。

命令

命令说明
python veil-sync.py --add <path>注册同步目标
python veil-sync.py --list列出已注册目标
python veil-sync.py --remove <path>取消注册目标
python veil-sync.py手动同步所有文件

支持的文件

工具文件
Claude CodeCLAUDE.md
CodexAGENTS.md
Cursor.cursorrules
GitHub Copilot.github/copilot-instructions.md
Gemini CLIGEMINI.md
Aider.aider.conf.yml

以下块将被插入配置文件中（.yml/.yaml 使用 # 格式）。

<!-- VEIL_START -->
Please follow these vocabulary rules in your output:
- close → complete
- branch → branch
<!-- VEIL_END -->

不转换的情况

以下模式会被有意跳过。

模式示例
文件名 / 路径组成部分（. 相邻）workflow_close.py
连字符分隔的标识符（- 相邻）task-close-workflow
反引号内部`close()`
双引号内部"close"
key=value 格式status=close
工单 ID（LETTERS-000）BRANCH-046
如果发生意外跳过，可以使用弹出窗口的"直接输入"手动转换。

类别

cat名称已转换用途
1描述性✓通用技术英语（close → 完成 等）
2固定 / 已排除—ALL_CAPS 常量、明确排除的词语
5专有名词 / 产品名✓工具名称、库名称
6项目专用✓项目专用术语
7模糊✓根据上下文需要转换的词语

输入原词时，类别会自动检测：

模式检测到的类别
ALL_CAPS、UPPER_CASE2（固定）
LETTERS-000（工单 ID）2（固定）
内部大写（camelCase、SQLite 等）5（专有名词）
其他英文单词 / 短语1（描述性）

Windows 自动启动

注册任务，使 VEIL 服务器在 Windows 登录时自动启动。

python install-startup.py           # 注册
python install-startup.py --remove  # 取消注册
python install-startup.py --status  # 检查状态

注册后，下次 Windows 登录时 VEIL 将自动启动。不会显示控制台窗口。
