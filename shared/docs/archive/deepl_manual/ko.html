DeepL API 키 발급 방법

VEIL 번역 후보 자동 생성 설정 가이드 — 약 5분 소요

DeepL API 무료 플랜 안내

DeepL API Free 플랜은 월 50만 자까지 무료로 이용할 수 있습니다. 신용카드 등록이 필요 없습니다. VEIL은 어휘 등록 시 후보 취득에만 사용하므로 일반적인 사용량에서는 무료 한도를 초과하지 않습니다.

발급 절차

  1. deepl.com/pro-api를 열고 「무료로 API 사용해 보기」 버튼을 클릭합니다.
  2. 이메일 주소와 비밀번호를 입력하여 계정을 생성합니다. 인증 메일이 도착하면 링크를 클릭하여 인증을 완료합니다.
  3. 로그인 후 우측 상단 계정 아이콘 → 「계정」 → 「API 키」 섹션으로 이동합니다. 표시된 API 키를 복사합니다.
  4. VEIL 프로젝트 디렉토리(veil/shared/) 안에 .env 파일을 생성하고 아래 내용을 저장합니다. 
    DEEPL_API_KEY=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx
    ※ 무료 플랜 키는 끝에 :fx가 붙습니다.
  5. VEIL 서버를 재시작합니다. 
    python app.py
    이후 어휘 등록 폼을 열면 번역 후보가 자동으로 입력됩니다.

동작 확인

사이드바의 「미변환어」 칩을 클릭하거나, 등록 폼의 「원어」 입력란에 영어 단어를 입력하세요. 「불러오는 중...」 표시 후 번역 후보가 자동 입력되면 설정 완료입니다.

API 키가 없어도 VEIL은 정상 동작합니다. 번역 후보 자동 입력이 되지 않을 뿐, 수동 입력은 그대로 사용할 수 있습니다.
