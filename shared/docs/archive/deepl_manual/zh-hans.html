如何获取 DeepL API 密钥

VEIL 自动翻译建议功能的设置指南 — 大约需要 5 分钟

关于 DeepL API 免费套餐

DeepL API Free 套餐每月可免费翻译 50 万个字符，无需绑定信用卡。 VEIL 仅在注册词汇时获取翻译建议，日常使用几乎不会超出免费额度。

获取步骤

  1. 打开 deepl.com/pro-api， 点击 「免费试用 API」 按钮。
  2. 输入电子邮件地址和密码创建账户。 收到验证邮件后，点击链接完成验证。
  3. 登录后，点击右上角账户图标 → 「账户」 → 「API 密钥」 部分。 复制显示的 API 密钥。
  4. 在 VEIL 项目目录（veil/shared/）中创建 .env 文件，写入以下内容并保存。 
    DEEPL_API_KEY=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx
    ※ 免费套餐的密钥末尾带有 :fx。
  5. 重新启动 VEIL 服务器。 
    python app.py
    之后打开词汇注册表单时，翻译建议将自动填入。

验证是否正常工作

点击侧边栏「未转换词」中的词汇标签，或在注册表单的「原词」栏输入英文单词。 显示「获取中...」后自动填入翻译建议，即表示设置成功。

即使没有 API 密钥，VEIL 也能正常运行。 只是无法自动填入翻译建议，手动输入功能仍然可以使用。
