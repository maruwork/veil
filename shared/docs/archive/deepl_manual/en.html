How to Get a DeepL API Key

Setup guide for VEIL's automatic translation suggestions — takes about 5 minutes

About the DeepL API Free Plan

The DeepL API Free plan allows up to 500,000 characters per month at no cost. No credit card is required. VEIL only uses the API when fetching translation suggestions during vocabulary registration, so typical usage stays well within the free limit.

Steps

  1. Open deepl.com/pro-api and click "Sign up for free".
  2. Enter your email address and create a password. Check your inbox for a confirmation email and click the verification link.
  3. After logging in, go to your account icon (top right) → Account → API Keys section. Copy the API key shown there.
  4. Create a .env file inside the veil/shared/ directory and add the following line: 
    DEEPL_API_KEY=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx
    Note: Free plan keys end with :fx.
  5. Restart the VEIL server: 
    python app.py
    Translation suggestions will now be auto-filled when you open the vocabulary registration form.

Verify It Works

Click any chip in the "Unmatched Words" section, or type an English word into the "Original" field of the registration form. If you see "Fetching..." followed by auto-filled suggestions, the setup is complete.

VEIL works without an API key. The only difference is that translation suggestions won't be auto-filled — you can still enter translations manually.
