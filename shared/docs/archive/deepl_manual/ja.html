DeepL APIキーの取得方法

VEIL で翻訳候補の自動生成を使うための設定手順（所要時間：約5分）

DeepL API 無料枠について

DeepL API Free プランは、月あたり 50万文字まで無料 で利用できます。 クレジットカードの登録は不要です。VEIL は語彙登録時の候補取得にのみ使用するため、 通常の使い方では無料枠を超えることはほとんどありません。

取得手順

  1. deepl.com/pro-api を開く。 ページ内の 「無料でAPIを試す」 または 「無料登録」 ボタンをクリック。
  2. メールアドレスとパスワードを入力してアカウントを作成する。 確認メールが届くので、リンクをクリックして認証を完了する。
  3. ログイン後、右上のアカウントアイコン → 「アカウント」 → 「APIキー」 セクションへ移動。 表示されている API キーをコピーする。
  4. VEIL のプロジェクトディレクトリ（veil/shared/）に .env ファイルを作成し、以下を記述して保存する。 
    DEEPL_API_KEY=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx
    ※ 無料プランのキーは末尾に :fx が付きます。
  5. VEIL サーバーを再起動する。 
    python app.py
    以降、語彙登録フォームを開いた際に翻訳候補が自動入力されます。

動作確認

サイドバーの「未変換語」チップをクリックするか、登録フォームの「元の語」に英単語を入力してください。 「生成中...」と表示されたあと翻訳候補が自動入力されれば設定完了です。

APIキーが設定されていない場合でも VEIL は動作します。 翻訳候補の自動入力が行われないだけで、手動入力は引き続き使えます。
