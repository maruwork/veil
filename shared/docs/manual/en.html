Overview

VEIL is a local tool that converts AI-generated English text into your preferred vocabulary. Conversion rules are stored in SQLite. DeepL free API is used for auto-generating translation candidates (API key is optional).

Startpython app.py → http://localhost:8080
RequirementsPython 3.8+, standard library only
Datavocab.db (SQLite, auto-created on first run)

Basic Usage

Text Input

Paste AI-generated text into the textarea at the top of the screen.

Running Conversion

  1. Paste text into the textarea
  2. Click Convert (or press Ctrl+Enter)
  3. The original text appears in the left column, converted text in the right column
  4. Converted words are highlighted in green

Changing Candidates

Click a highlighted word to open the popup.

ActionDescription
Select candidateUse that translation. The selected word becomes p1 (primary candidate) next time
Custom inputType directly and confirm
RevertUndo conversion and restore the original English (DB not changed)
ExcludeExclude this word from conversion. Skipped from now on

Vocabulary Registration

Form Registration

Use the "Add Vocabulary" form in the top of the sidebar.

FieldDescription
Original (English)The English word or phrase to convert. Required
Translation 1Primary translation
Reading (optional)Phonetic or alternate reading
Translation 2 (optional)Secondary candidate
CategorySee the Categories section

The category is auto-detected when you type the original word. If a DeepL API key is set, translation candidates are auto-filled when opening the form from a popup or unmatched word chip.

Add / Update briefly turns the button green to confirm registration. Re-registering the same word performs an upsert (overwrite update).

Quick Registration (Text Selection)

  1. Drag to select text in the converted output
  2. A + Register button floats near the selection
  3. Click it to auto-fill the form and fetch DeepL candidates

From Unmatched Words

After conversion, English words that were not converted appear as chips in the "Unmatched Words" section of the sidebar (up to 10, sorted by frequency). Click a chip to auto-fill the registration form.

Stop words (a, the, is, etc.) and already-registered words are excluded.

The "Registered" list in the bottom of the sidebar lets you view and manage vocabulary.

Export / Import

Back up and migrate vocabulary data using the buttons in the bottom right of the footer.

Language Pairs and Data Isolation

Vocabulary data is managed independently per language pair. EN→JA and EN→KO data are stored in the same vocab.db file but never interfere with each other. Both Export and Import target only the currently selected language pair.

Export

Downloads all vocabulary registered for the current language pair as a JSON file.

Filename: veil-vocab-YYYYMMDD.json

Import

Select a JSON file to load. Existing vocabulary is upserted (overwrite updated). Nothing is deleted. Imported vocabulary is registered to the current language pair.

To import data for a different language, switch the language selector in the header first, then Import.
The only required field in the import file is o (original word). Words without p1 are registered with no translation.

veil-sync — Sync to Other AI Tools

Automatically inserts registered vocabulary into CLAUDE.md, AGENTS.md, .cursorrules and other config files. Syncs in the background whenever vocabulary is added or deleted.

Initial Setup

python veil-sync.py --add /path/to/CLAUDE.md

Syncs immediately on registration. Repeat for multiple files.

Commands

CommandDescription
python veil-sync.py --add <path>Register a sync target
python veil-sync.py --listList registered targets
python veil-sync.py --remove <path>Unregister a target
python veil-sync.pyManually sync all files

Supported Files

The following block is inserted into config files (.yml/.yaml use # format).

<!-- VEIL_START -->
Please follow these vocabulary rules in your output:
- close → complete
- branch → branch
<!-- VEIL_END -->

Cases Not Converted

The following patterns are intentionally skipped.

PatternExample
Filename / path component (. adjacent)workflow_close.py
Hyphen-delimited identifier (- adjacent)task-close-workflow
Inside backticks`close()`
Inside double quotes"close"
key=value formatstatus=close
Ticket ID (LETTERS-000)BRANCH-046
If an unexpected skip occurs, you can manually convert using the "Custom input" popup.

Categories

catNameConvertedUse
1Descriptive✓General technical English (close → complete, etc.)
2Fixed / Excluded—ALL_CAPS constants, explicitly excluded words
5Proper noun / Product✓Tool names, library names
6Project-specific✓Project-specific terminology
7Ambiguous✓Words that need conversion depending on context

Category is auto-detected when you type the original word:

PatternDetected Category
ALL_CAPS, UPPER_CASE2 (Fixed)
LETTERS-000 (ticket ID)2 (Fixed)
Internal capitals (camelCase, SQLite, etc.)5 (Proper noun)
Other English words / phrases1 (Descriptive)

Windows Auto-Start

Register a task to automatically start the VEIL server when Windows logs in.

python install-startup.py           # Register
python install-startup.py --remove  # Unregister
python install-startup.py --status  # Check status

After registration, VEIL starts automatically at the next Windows login. No console window is shown.
