概述

VEIL 是一個本地工具，可將 AI 生成的英文文字轉換為您偏好的詞彙。轉換規則儲存在 SQLite 中。自動生成翻譯候選詞使用 DeepL 免費 API（API 金鑰為選用項目）。

啟動python app.py → http://localhost:8080
要求Python 3.8 以上，僅使用標準函式庫
資料vocab.db（SQLite，首次執行時自動建立）

基本用法

文字輸入

將 AI 生成的文字貼到畫面上方的文字方塊中。

執行轉換

  1. 將文字貼到文字方塊中
  2. 點擊轉換（或按 Ctrl+Enter）
  3. 左欄顯示原文，右欄顯示轉換後的文字
  4. 已轉換的詞語以綠色醒目顯示

變更候選詞

點擊醒目顯示的詞語即可開啟快顯視窗。

操作說明
選擇候選詞使用該翻譯。下次所選詞語將成為 p1（首選候選詞）
直接輸入直接輸入並確認
還原取消轉換並恢復原始英文（不更改資料庫）
排除將該詞從轉換中排除，此後將略過

詞彙登錄

表單登錄

使用側邊欄上方的「新增詞彙」表單。

欄位說明
原詞（英文）要轉換的英文單字或片語。必填
譯文 1主要翻譯
讀音（選用）發音或備用讀法
譯文 2（選用）第二候選詞
類別請參閱類別部分

輸入原詞時，類別會自動偵測。如果設定了 DeepL API 金鑰，從快顯視窗或未轉換詞標籤開啟表單時，翻譯候選詞將自動填入。

新增 / 更新按鈕會短暫變為綠色以確認登錄。重新登錄相同單字將執行 upsert（覆寫更新）。

快速登錄（文字選取）

  1. 在轉換後的輸出中拖曳選取文字
  2. 選取範圍附近會出現+ 登錄按鈕
  3. 點擊即可自動填寫表單並取得 DeepL 候選詞

從未轉換詞登錄

轉換後，未被轉換的英文詞語會以標籤形式顯示在側邊欄的「未轉換詞」部分（最多 10 個，按頻率排序）。點擊標籤即可自動填寫登錄表單。

停用詞（a、the、is 等）和已登錄詞語將被排除。

已登錄詞彙

側邊欄下方的「已登錄」清單可查看和管理詞彙。

操作說明
搜尋框依原詞或譯文 1 篩選。最多顯示 20 筆
類別篩選僅顯示特定類別
頻率 / 順序按鈕切換排序方式
× 按鈕刪除詞彙項目

清單說明

狀態含義
正常（不透明）轉換中使用的作用中項目
暗淡（半透明）cat=2（已排除）或無翻譯。不用於轉換
（無翻譯）屬於目標類別但未輸入翻譯。用作預留位置或等待輸入

匯出 / 匯入

使用頁尾右下角的按鈕備份和移轉詞彙資料。

語言對與資料隔離

詞彙資料按語言對獨立管理。 EN→JA 和 EN→KO 的資料儲存在同一個 vocab.db 中，但互不干擾。 匯出和匯入均僅針對目前選取的語言對。

匯出

將目前語言對中已登錄的所有詞彙下載為 JSON 檔案。

檔案名稱: veil-vocab-YYYYMMDD.json

匯入

選取 JSON 檔案載入。現有詞彙將執行 upsert（覆寫更新）。不會刪除任何內容。匯入的詞彙將登錄至目前的語言對。

若要匯入其他語言的資料，請先切換標題列的語言選取器，再執行匯入。
匯入檔案中唯一必填欄位是 o（原詞）。沒有 p1 的詞語將以無翻譯方式登錄。

veil-sync — 與其他 AI 工具同步

自動將已登錄詞彙插入 CLAUDE.md、AGENTS.md、.cursorrules 等設定檔。每次新增或刪除詞彙時都會在背景同步。

初始設定

python veil-sync.py --add /path/to/CLAUDE.md

登錄後立即同步。對多個檔案重複操作。

指令

指令說明
python veil-sync.py --add <path>登錄同步目標
python veil-sync.py --list列出已登錄目標
python veil-sync.py --remove <path>取消登錄目標
python veil-sync.py手動同步所有檔案

支援的檔案

工具檔案
Claude CodeCLAUDE.md
CodexAGENTS.md
Cursor.cursorrules
GitHub Copilot.github/copilot-instructions.md
Gemini CLIGEMINI.md
Aider.aider.conf.yml

以下區塊將被插入設定檔中（.yml/.yaml 使用 # 格式）。

<!-- VEIL_START -->
Please follow these vocabulary rules in your output:
- close → complete
- branch → branch
<!-- VEIL_END -->

不轉換的情況

以下模式會被刻意略過。

模式範例
檔案名稱 / 路徑組成部分（. 相鄰）workflow_close.py
連字號分隔的識別項（- 相鄰）task-close-workflow
反引號內部`close()`
雙引號內部"close"
key=value 格式status=close
票券 ID（LETTERS-000）BRANCH-046
如果發生意外略過，可以使用快顯視窗的「直接輸入」手動轉換。

類別

cat名稱已轉換用途
1描述性✓通用技術英語（close → 完成 等）
2固定 / 已排除—ALL_CAPS 常數、明確排除的詞語
5專有名詞 / 產品名稱✓工具名稱、函式庫名稱
6專案專用✓專案專用術語
7模糊✓根據上下文需要轉換的詞語

輸入原詞時，類別會自動偵測：

模式偵測到的類別
ALL_CAPS、UPPER_CASE2（固定）
LETTERS-000（票券 ID）2（固定）
內部大寫（camelCase、SQLite 等）5（專有名詞）
其他英文單字 / 片語1（描述性）

Windows 自動啟動

登錄工作，使 VEIL 伺服器在 Windows 登入時自動啟動。

python install-startup.py           # 登錄
python install-startup.py --remove  # 取消登錄
python install-startup.py --status  # 確認狀態

登錄後，下次 Windows 登入時 VEIL 將自動啟動。不會顯示主控台視窗。
