개요

VEIL은 AI가 생성한 영어 텍스트를 원하는 어휘로 변환하는 로컬 도구입니다. 변환 규칙은 SQLite에 저장됩니다. 번역 후보 자동 생성에는 DeepL 무료 API를 사용합니다(API 키는 선택 사항).

시작python app.py → http://localhost:8080
요구 사항Python 3.8 이상, 표준 라이브러리만 사용
데이터vocab.db (SQLite, 최초 실행 시 자동 생성)

기본 사용법

텍스트 입력

화면 상단의 텍스트 영역에 AI가 생성한 텍스트를 붙여넣습니다.

변환 실행

  1. 텍스트 영역에 텍스트를 붙여넣습니다
  2. 변환을 클릭하거나 Ctrl+Enter를 누릅니다
  3. 왼쪽 칸에 원문, 오른쪽 칸에 변환된 텍스트가 표시됩니다
  4. 변환된 단어는 녹색으로 강조 표시됩니다

후보 변경

강조 표시된 단어를 클릭하면 팝업이 열립니다.

동작설명
후보 선택해당 번역을 사용합니다. 선택한 단어는 다음 번에 p1(첫 번째 후보)이 됩니다
직접 입력직접 입력하여 확인합니다
되돌리기변환을 취소하고 원래 영어로 복원합니다(DB는 변경되지 않음)
제외이 단어를 변환에서 제외합니다. 이후로는 건너뜁니다

어휘 등록

폼 등록

사이드바 상단의 "어휘 추가" 폼을 사용합니다.

항목설명
원어(영어)변환할 영어 단어 또는 구문. 필수
번역 1기본 번역
읽기(선택)발음 또는 대체 읽기
번역 2(선택)두 번째 후보
카테고리카테고리 섹션을 참조하세요

원어를 입력하면 카테고리가 자동으로 감지됩니다. DeepL API 키가 설정되어 있으면 팝업이나 미변환어 칩에서 폼을 열 때 번역 후보가 자동으로 채워집니다.

추가 / 업데이트 버튼이 잠시 녹색으로 변하며 등록을 확인합니다. 같은 단어를 다시 등록하면 upsert(덮어쓰기 업데이트)가 수행됩니다.

빠른 등록 (텍스트 선택)

  1. 변환된 출력에서 텍스트를 드래그하여 선택합니다
  2. 선택 영역 근처에 + 등록 버튼이 표시됩니다
  3. 클릭하면 폼이 자동으로 채워지고 DeepL 후보를 가져옵니다

미변환어에서 등록

변환 후, 변환되지 않은 영어 단어가 사이드바의 "미변환어" 섹션에 칩으로 표시됩니다(최대 10개, 빈도순). 칩을 클릭하면 등록 폼이 자동으로 채워집니다.

불용어(a, the, is 등)와 이미 등록된 단어는 제외됩니다.

등록된 어휘

사이드바 하단의 "등록됨" 목록에서 어휘를 확인하고 관리할 수 있습니다.

동작설명
검색창원어 또는 번역 1로 필터링. 최대 20개 표시
카테고리 필터특정 카테고리만 표시
빈도 / 순서 버튼정렬 순서 전환
× 버튼어휘 항목 삭제

목록 보는 법

상태의미

내보내기 / 가져오기

푸터 오른쪽 하단의 버튼을 사용하여 어휘 데이터를 백업하고 마이그레이션합니다.

언어 쌍과 데이터 분리

어휘 데이터는 언어 쌍별로 독립적으로 관리됩니다. EN→JA와 EN→KO 데이터는 동일한 vocab.db에 저장되지만 서로 간섭하지 않습니다. 내보내기와 가져오기는 모두 현재 선택된 언어 쌍만을 대상으로 합니다.

내보내기

현재 언어 쌍에 등록된 모든 어휘를 JSON 파일로 다운로드합니다.

파일명: veil-vocab-YYYYMMDD.json

가져오기

JSON 파일을 선택하여 로드합니다. 기존 어휘는 upsert(덮어쓰기 업데이트)됩니다. 삭제는 되지 않습니다. 가져온 어휘는 현재 언어 쌍에 등록됩니다.

다른 언어의 데이터를 가져오려면 먼저 헤더의 언어 선택기를 전환한 후 가져오기를 실행하세요.
가져오기 파일에서 필수 항목은 o(원어)뿐입니다. p1이 없는 단어는 번역 없이 등록됩니다.

veil-sync — 다른 AI 도구와 동기화

등록된 어휘를 CLAUDE.md, AGENTS.md, .cursorrules 등의 설정 파일에 자동으로 삽입합니다. 어휘를 추가하거나 삭제할 때마다 백그라운드에서 동기화됩니다.

초기 설정

python veil-sync.py --add /path/to/CLAUDE.md

등록 즉시 동기화됩니다. 여러 파일에 반복하세요.

명령어

명령어설명
python veil-sync.py --add <path>동기화 대상 등록
python veil-sync.py --list등록된 대상 목록 표시
python veil-sync.py --remove <path>대상 등록 해제
python veil-sync.py모든 파일 수동 동기화

변환되지 않는 경우

다음 패턴은 의도적으로 건너뜁니다.

패턴예시
파일명 / 경로 구성 요소(. 인접)workflow_close.py
하이픈 구분 식별자(- 인접)task-close-workflow
백틱 내부`close()`
큰따옴표 내부"close"
key=value 형식status=close
티켓 ID (LETTERS-000)BRANCH-046
예상치 못한 건너뜀이 발생하면 팝업의 "직접 입력"으로 수동 변환할 수 있습니다.

카테고리

cat이름변환됨용도
1설명적✓일반 기술 영어 (close → 완료 등)
2고정 / 제외됨—ALL_CAPS 상수, 명시적으로 제외된 단어
5고유명사 / 제품명✓도구 이름, 라이브러리 이름
6프로젝트 고유✓프로젝트 고유 용어
7모호함✓문맥에 따라 변환이 필요한 단어

원어를 입력하면 카테고리가 자동으로 감지됩니다:

LETTERS-000 (티켓 ID)2 (고정)

Windows 자동 시작

Windows 로그인 시 VEIL 서버가 자동으로 시작되도록 작업을 등록합니다.

python install-startup.py           # 등록
python install-startup.py --remove  # 등록 해제
python install-startup.py --status  # 상태 확인

등록 후 다음 Windows 로그인 시 VEIL이 자동으로 시작됩니다. 콘솔 창은 표시되지 않습니다.
