如何取得 DeepL API 金鑰

VEIL 自動翻譯建議功能的設定指南 — 約需 5 分鐘

關於 DeepL API 免費方案

DeepL API Free 方案每月可免費翻譯 50 萬個字元，無需綁定信用卡。 VEIL 僅在登錄詞彙時取得翻譯建議，一般使用幾乎不會超出免費額度。

取得步驟

  1. 開啟 deepl.com/pro-api， 點擊 「免費試用 API」 按鈕。
  2. 輸入電子郵件地址和密碼建立帳號。 收到驗證郵件後，點擊連結完成驗證。
  3. 登入後，點擊右上角帳號圖示 → 「帳號」 → 「API 金鑰」 區塊。 複製顯示的 API 金鑰。
  4. 在 VEIL 專案目錄（veil/shared/）中建立 .env 檔案，寫入以下內容並儲存。 
    DEEPL_API_KEY=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx
    ※ 免費方案的金鑰末尾帶有 :fx。
  5. 重新啟動 VEIL 伺服器。 
    python app.py
    之後開啟詞彙登錄表單時，翻譯建議將自動填入。

確認是否正常運作

點擊側邊欄「未轉換詞」中的詞彙標籤，或在登錄表單的「原詞」欄位輸入英文單字。 顯示「獲取中...」後自動填入翻譯建議，即表示設定成功。

即使沒有 API 金鑰，VEIL 也能正常運作。 只是無法自動填入翻譯建議，手動輸入功能仍然可以使用。
