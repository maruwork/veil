# PS Suite Idea Diverge Pack

アイデア発散フェーズの reusable baseline pack。

## 1. Purpose

- 評価を保留したまま量を出す
- 視点を強制的に切り替えて候補を広げる

## 2. Minimum Contract

- input fields: theme / domain / constraints / exclusions
- multiple ideation lenses
- bridge summary to next phase

## 3. Baseline Techniques

- brainstorming
- SCAMPER
- reverse thinking
- cross-industry combination
- Osborne checklist

## 4. Completion Rule

- candidate set が十分広がっている
- top candidates と bridge summary が残っている
- converge phase へ渡す情報が揃っている
