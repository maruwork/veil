# PS Suite Idea Diverge Pack

Reusable baseline pack for the idea divergence phase.

## 1. Purpose

- Generate volume before evaluation starts
- Force perspective shifts so the candidate space expands

## 2. Minimum Contract

- input fields: theme / domain / constraints / exclusions
- multiple ideation lenses
- bridge summary to the next phase

## 3. Baseline Techniques

- brainstorming
- SCAMPER
- reverse thinking
- cross-industry combination
- Osborne checklist

## 4. Completion Rule

- the candidate set is broad enough
- top candidates and a bridge summary remain
- the next phase has enough information to start convergence
