# PS Suite Idea Converge Pack

Reusable baseline pack for the idea convergence phase.

## 1. Purpose

- Turn a wide candidate set into a comparable shortlist
- Compress the shortlist to three items or fewer

## 2. Minimum Contract

- input shortlist
- grouping / evaluation / prioritization methods
- selection rationale
- bridge summary to the sharpen phase

## 3. Baseline Techniques

- KJ grouping
- Six Thinking Hats
- priority matrix
- weakness-to-improvement review

## 4. Completion Rule

- shortlisted ideas are explicit
- ranking and rationale are recorded
- a bridge summary for the sharpen phase is present
