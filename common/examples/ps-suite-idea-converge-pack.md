# PS Suite Idea Converge Pack

アイデア収束フェーズの reusable baseline pack。

## 1. Purpose

- 発散した候補を比較可能な形へ揃える
- shortlist を 3 件以内まで圧縮する

## 2. Minimum Contract

- input shortlist
- grouping / evaluation / prioritization methods
- selection reason
- sharpen phase への bridge summary

## 3. Baseline Techniques

- KJ grouping
- Six Thinking Hats
- priority matrix
- weakness-to-improvement review

## 4. Completion Rule

- shortlisted ideas が明示されている
- ranking と理由がある
- sharpen phase に渡す bridge summary がある
