# PS Suite Idea Sharpen Pack

アイデア具体化フェーズの reusable baseline pack。

## 1. Purpose

- shortlist を実行可能な候補へ具体化する
- go/no-go の材料を揃える

## 2. Minimum Contract

- target idea
- constraints
- decision priority
- value / execution / risk / persona / go-no-go review

## 3. Baseline Techniques

- full idea expansion
- risk analysis
- persona analysis
- improvement paths
- go/no-go scoring

## 4. Completion Rule

- execution-ready summary がある
- main risks と mitigations がある
- go/no-go か conditional go/no-go が判断できる
