# PS Suite Idea Sharpen Pack

Reusable baseline pack for the idea sharpening phase.

## 1. Purpose

- Turn the shortlist into executable candidates
- Gather enough material for a go / no-go decision

## 2. Minimum Contract

- target idea
- constraints
- decision priority
- value / execution / risk / persona / go-no-go review

## 3. Baseline Techniques

- full idea expansion
- risk analysis
- persona analysis
- improvement paths
- go / no-go scoring

## 4. Completion Rule

- an execution-ready summary exists
- main risks and mitigations are recorded
- go / no-go or conditional go / no-go can be judged
