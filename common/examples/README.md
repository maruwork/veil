# Shared Examples

This shelf stores small examples for shared policies, templates, checklists, and frameworks.

Examples should stay minimal and portable.
Treat this shelf as support only, not first-read material.
Project-specific examples belong in the relevant project shelf.
Use this shelf only after you already know which rule, template, checklist, or framework you are trying to apply.
