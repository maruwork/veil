# Decision Packet Template

**When to use**: Use this when autonomous work must stop because final human judgment is required.  
**Recommended filename**: `YYYYMMDD_<topic>-decision-packet.md`  
**Replace**: task ID format, reviewer name, post-stop return route, number of options.  
**Do not write**: the implementation plan itself, the current-state canonical source, or a redefinition of the project board.

Use this packet only when real owner judgment is required.
If work can continue without that judgment, update the current surface and continue instead.
