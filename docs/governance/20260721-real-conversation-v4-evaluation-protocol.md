# VEIL real-conversation v4 independent evaluation protocol

v4 uses fresh, anonymized real-conversation excerpts that do not repeat v1,
v2, or v3 excerpts. It applies the v2 candidate-eligibility clarification and
the v3 primary-target clarification.

Reviewers select each primary lexical target whose meaning, scope, allowed use,
or policy is explicitly decided. A generic explanatory predicate is evidence,
not a target. Several independent targets are allowed in one session; duplicate
normalized targets are not. Ordinary operational process and state mentions are
excluded.

Two independent reviewers validate source fidelity, canonical-snapshot
membership, eligibility, primary target choice, and term uniqueness. The blind
generator sees only frozen runtime text and semantic-frame instructions. First
execution is immutable. Artifacts stay in
`workspace/audit/20260721-real-conversation-ux-v4/` for 30 days. DB, Git,
install, sync, remote, and source implementation writes are prohibited.
