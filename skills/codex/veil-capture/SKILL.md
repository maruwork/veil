# veil-capture — チャットからVEILルールを抽出

会話テキストを解析し、AIが使った英語・造語・略語を検出して日本語に変換し、`~/.veil/rules/` に書き込む。

## 手順

### 1. 解析対象テキストの決定

- 引数が指定されている場合 → その文字列を解析対象とする
- 指定がない場合 → この会話全体を解析対象とする（ユーザー発言・AI発言の両方を含む）

### 2. 検出

以下を拾う：

- 英語のまま使った語（日本語・カタカナで表記できるもの）
- 独自造語・複合語
- 略語（業界標準でないもの）

**除外する文脈：**

- バッククォート（`` ` ``）で囲まれた語 → コード識別子・ファイルパスとして除外
- ダブルクォートで囲まれた語 → 固有値として除外
- `key=value` パターン → 設定値として除外
- 固有名詞（ツール名・製品名・サービス名）→ 除外

### 3. ファイルへ書き込む

#### ファイルテンプレート

```markdown
# {letter}

- {term} → {訳語}
```

- ヘッダー `# {letter}` は先頭文字（小文字）
- エントリは `- {元語} → {推奨表記}` の形式で1行1エントリ
- エントリは元語のアルファベット昇順に並べる

#### 書き込み先

| 条件 | ファイル |
|------|---------|
| a〜z で始まる語 | `~/.veil/rules/{先頭文字}.md` |
| 数字・記号で始まる語 | `~/.veil/rules/special.md` |

先頭文字は小文字に統一。

#### 書き込み手順

1. 対象ファイルを読み込む（存在しない場合はテンプレートから新規作成）
2. 既存エントリと **大文字小文字を区別せずに** 元語を比較し、一致する場合はスキップ
3. 新規エントリを追加し、全エントリを元語の昇順（case-insensitive）で並べ替えて上書き保存

### 5. veil-sync を実行

ファイル書き込み後、`veil-sync.py` を実行して同期先ファイルに即時反映する。

`~/.veil/config.json` の `sync_script` キーからパスを取得する。

```bash
# bash / zsh (macOS / Linux)
python "$(python3 -c "import json; print(json.load(open('$HOME/.veil/config.json'))['sync_script'])")"
```

```powershell
# PowerShell (Windows)
python (python -c "import json,os; print(json.load(open(os.path.expanduser('~/.veil/config.json')))['sync_script'])")
```

`config.json` が存在しない、または `sync_script` キーがない場合はユーザーに場所を確認する。
VEILサーバーが起動していない場合は同期をスキップし、その旨を報告する。

### 6. 完了報告

書き込んだ内容を表形式で報告する（追記件数・スキップ件数・ファイルパス・sync結果を含む）。

| 元語 | 推奨表記 | 書き込み先 | 結果 |
|------|---------|----------|------|
| repo | リポジトリ | `~/.veil/rules/r.md` | 追記 |
| ... | ... | ... | スキップ（既存） |
