## Issues

Open a GitHub issue for bugs, documentation gaps, or feature requests.

Use the bug report template for reproducible defects. Use a feature request issue for workflow or skill improvements.

## Security

Do not open public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).

## Scope

VEIL is a local vocabulary guardrail. It does not call external LLM APIs. Support covers the CLI tools, installers, and veil-capture skills shipped in this repository.
