# /veil-capture — チャットからVEILルールを抽出

この会話を遡り、AIが使った英語・造語・略語・カタカナで書けるのに英語にした語を検出する。
検出した語をLLMの判断で日本語に変換し、`~/.veil/rules/` に書き込む。

## 手順

### 1. 検出

会話全体を対象に、以下を拾う：

- AIが英語のまま使った語（日本語・カタカナで表記できるもの）
- AIが独自に作った造語・複合語
- 略語（業界標準でないもの）

### 2. 変換候補を提示

検出した語を表形式で提示し、ユーザーに確認を求める。

| 元語 | 推奨表記 | 種別 |
|------|---------|------|
| workflow | ワークフロー | カタカナ化 |
| ... | ... | ... |

種別: カタカナ化 / 日本語訳 / 造語削除 / 略語展開

### 3. 確認後にファイルへ書き込む

#### ファイルテンプレート

```markdown
# {letter}

- {term} → {訳語}
```

- ヘッダー `# {letter}` は先頭文字（小文字）
- エントリは `- {元語} → {推奨表記}` の形式で1行1エントリ
- エントリは元語のアルファベット昇順に並べる

#### 書き込み先

| 条件 | ファイル |
|------|---------|
| a〜z で始まる語 | `~/.veil/rules/{先頭文字}.md` |
| 数字・記号で始まる語 | `~/.veil/rules/special.md` |

先頭文字は小文字に統一。

#### 書き込み手順

1. 対象ファイルを読み込む（存在しない場合はテンプレートから新規作成）
2. 同じ元語（`- {元語} →`）が既にあればスキップ
3. 新規エントリを追加し、全エントリを元語の昇順で並べ替えて上書き保存

### 4. veil-sync を実行

ファイル書き込み後、`veil-sync.py` を実行して同期先ファイルに即時反映する。

`~/.veil/config.json` の `sync_script` キーからパスを取得する。

```bash
python $(cat ~/.veil/config.json | python3 -c "import sys,json; print(json.load(sys.stdin)['sync_script'])")
```

`config.json` が存在しない、または `sync_script` キーがない場合はユーザーに場所を確認する。
VEILサーバーが起動していない場合は同期をスキップし、その旨を報告する。

### 5. 完了報告

追記した件数（スキップした件数も含む）、書き込んだファイルパス、sync結果を報告する。
