# /veil-capture — チャットからVEILルールを抽出

この会話を遡り、AIが使った英語・造語・略語・カタカナで書けるのに英語にした語を検出する。
検出した語をLLMの判断で日本語に変換し、`~/.veil/rules/` に追記する。

## 手順

### 1. 検出

会話全体を対象に、以下を拾う：

- AIが英語のまま使った語（日本語・カタカナで表記できるもの）
- AIが独自に作った造語・複合語
- 略語（業界標準でないもの）

### 2. 変換候補を提示

検出した語を表形式で提示し、ユーザーに確認を求める。

| 元語 | 推奨表記 | 種別 |
|------|---------|------|
| workflow | ワークフロー | カタカナ化 |
| ... | ... | ... |

種別: カタカナ化 / 日本語訳 / 造語削除 / 略語展開

### 3. 確認後にファイルへ追記

ユーザーが確認・修正したら、元語の**先頭文字**に対応するファイルへ書き込む。

- 書き込み先: `~/.veil/rules/{先頭文字}.md`
  - 例: `workflow` → `~/.veil/rules/w.md`
  - 数字・記号で始まる語 → `~/.veil/rules/special.md`
  - 先頭文字は小文字に統一
- ディレクトリが存在しない場合は作成する
- ファイルが存在しない場合は新規作成する
- **書き込み前に対象ファイルを読み込み、同じ元語（`- {元語} →`）が既にあればスキップする**

追記フォーマット：
```
- workflow → ワークフロー
- deploy → デプロイ
```

### 4. 完了報告

追記した件数（スキップした件数も含む）と書き込んだファイルパスを報告する。
次回の `veil-sync.py` 実行時に自動的に同期先ファイルへ反映される旨を添える。
