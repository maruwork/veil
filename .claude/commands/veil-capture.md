# /veil-capture — チャットからVEILルールを抽出

この会話を遡り、AIが使った英語・造語・略語・カタカナで書けるのに英語にした語を検出する。
検出した語をLLMの判断で日本語に変換し、`~/.veil/base-rules.md` に追記する。

## 手順

### 1. 検出

会話全体を対象に、以下を拾う：

- AIが英語のまま使った語（日本語・カタカナで表記できるもの）
- AIが独自に作った造語・複合語
- 略語（業界標準でないもの）
- 翻訳調の表現（英語の文章構造をそのまま日本語にしたもの）

VEILの語彙DBに既に登録されている語は除外しない（重複してもよい）。

### 2. 変換候補を提示

検出した語を表形式で提示し、ユーザーに確認を求める。

| 元語 | 推奨表記 | 種別 |
|------|---------|------|
| workflow | ワークフロー | カタカナ化 |
| ... | ... | ... |

種別: カタカナ化 / 日本語訳 / 造語削除 / 略語展開

### 3. 確認後にファイルへ追記

ユーザーが確認・修正したら、`~/.veil/base-rules.md` に追記する。

追記フォーマット：
```
- workflow → ワークフロー
- deploy → デプロイ
```

ファイルが存在しない場合は新規作成する。
既存の内容は保持し、重複する元語があれば上書きせず末尾に追加する（重複排除はユーザーが手動で行う）。

### 4. 完了報告

追記した件数と `~/.veil/base-rules.md` のパスを報告する。
次回の `veil-sync.py` 実行時に自動的に同期先ファイルへ反映される旨を添える。
